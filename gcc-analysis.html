Guardian Cryptic Crossword Analysis

This page renders some charts from data gathered via the Guardian newspaper's cryptic crosswords. The data shown here was scraped via the web.

The git repository containing this data is here.

1. Number of crosswords set per setter

This chart shows the number of crosswords set per setter. No real surprises here as to the most prolific setters.


2. Crosswords per year, per setter

This chart shows an area span for the number of crosswords set per setter, per year. Interesting to see when a setter started and stopped.


3. Frequency of word duplications across all crosswords, per setter

This chart shows the number of words a given setter has used more than once, across all the crosswords that setter has set.
